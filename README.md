# DOKOMIRU(仮)- アニメ配信検索サイト

アニメタイトルから、正規配信サービス(Abema, Netflix, dアニメストア等)をまとめて検索できるサイトの土台です。

## 仕組み

- `schema.sql` … データベースの設計図(アニメ、配信サービス、どのアニメがどこで見れるかの3つの表)
- `seed.py` … データベースを初期化し、サンプルデータを入れるスクリプト
- `app.py` … Flaskサーバー本体。URLは `/anime/<slug>` の1パターンだけで、
  データベースの中身に応じて全作品のページを自動生成する
- `templates/` … ページの見た目(base.html が共通レイアウト、他は中身)
- `static/style.css` … デザイン

## 使い方(ローカルで動かす)

```bash
pip install flask
python seed.py      # データベースを作る(初回のみ、またはデータを増やすたびに再実行可)
python app.py        # サーバー起動 → http://localhost:5001
```

## 【重要】このデータについて

`seed.py` には、2026年7月時点でWeb検索により確認した**実在の人気アニメ5作品**
(鬼滅の刃、呪術廻戦、SPY×FAMILY、葬送のフリーレン、ONE PIECE)のデータが
入っています。ただし配信権は今後も変わる可能性があるため、公開前および
定期的に、必ず公式サイトで最新情報を確認し、内容を更新してください。
各作品ページの「最終確認日」は、あくまでこのデータを投入した日付の目安です。

また、視聴リンク(`watch_url`)は現時点では各サービスのトップページに
なっています。作品ごとの視聴ページへの直接リンクが分かれば、
`seed.py` 内の該当箇所を書き換えることでユーザー体験が向上します。

## 新しいアニメを1本追加する方法

`seed.py` の `ANIME` リストに、以下の形式でオブジェクトを追加するだけです。
ページは自動的に `/anime/(slugで指定した文字列)` として生成されます。

```python
{
    "slug": "example-title",             # URLになる(英数字とハイフンのみ推奨)
    "title": "作品名",
    "title_kana": "さくひんめい",         # 検索用
    "synopsis": "あらすじ文章",
    "episode_count": 12,
    "release_year": 2026,
    "genre": "ジャンル1,ジャンル2",
    "is_airing": 0,                       # 放送中なら1
    "availability": [
        {"service": "Abema", "plan_type": "free", "watch_url": "実際の視聴URL"},
    ],
},
```

追加したら `python seed.py` を再実行すれば反映されます。
